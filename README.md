# Showy

Showy is a VS Code extension that visualizes the current workspace as an interactive tree. It shows folders and files as clickable nodes and displays stats for the selected file or folder.

## Usage
1. Open the `showy` folder in VS Code.
2. Run `npm install`.
3. Run `npm run compile`.
4. Press `F5` to launch the extension in a new Extension Development Host.
5. Use the `Showy: Open Project Map` command from the command palette.
